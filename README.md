# SNGR-CZ5-ENOS v0.6.4

Versión de revisión enfocada en el botón **Riesgo Zonal**, particularmente en el enfoque **Análisis** y en la proyección de geometrías del formulario **F03**.

## Incluye

- Ficha cantonal de Análisis con criterios técnicos diferenciados.
- Bloque Cartografía F03 para puntos, líneas y polígonos.
- Filtros F03 en el mapa.
- Botón para abrir Kobo F03.
- Dashboard actualizado con avance de integración Riesgo Zonal/F03.
- Esquemas JSON para análisis técnico y cartografía F03.

## Nota

Los elementos F03 incluidos son demostrativos y no oficiales. La conexión real debe hacerse desde la exportación Kobo F03, ArcGIS Online o un GeoJSON validado.
